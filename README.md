# Progressive Times News App

> A demonstration of a Progressive Web App
>- Progressive Web Apps (PWA) run in the browser on a mobile device and act just like a native app.
>- Caching
>- Service Workers
>- Fetch API
>- Manifest file
>- Add to Home Screen
>- Push notifications
>- Offline browsing
>- Background Sync

- Work in Progress

## Dependencies

- JavaScript
